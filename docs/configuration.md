# Configuración

1. Ingrese en **Keygin Sync → Configuración**.
2. Pegue sus credenciales de Contifico:
   - API Key
   - Email
   - Empresa
3. Presione **Verificar conexión**.
4. Seleccione la bodega principal.
5. Guarde los cambios.

El sistema comenzará a sincronizar productos y stock.
