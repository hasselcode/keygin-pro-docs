# Licencias

Keygin ERP Sync Pro funciona bajo un modelo anual:

- 1 licencia = 1 sitio
- Renovación anual
- Trial gratuito de 7 días

## Activación

1. Ingrese en **Keygin Sync → Licencia**.
2. Pegue su clave de licencia.
3. Presione **Activar**.

Si la licencia es válida, el sistema quedará habilitado.
