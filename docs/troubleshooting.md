# Solución de problemas

## Error de conexión

- Verifique que su API Key sea correcta.
- Confirme que Contifico esté operativo.

## Productos no sincronizan

- Revise que exista una bodega seleccionada.
- Confirme que los SKUs estén definidos.

## Logs

Los registros se encuentran en:

WordPress → Herramientas → Logs de Keygin
