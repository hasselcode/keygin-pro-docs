# Keygin ERP Sync Pro

Documentación oficial de **Keygin ERP Sync Pro**, la solución profesional de sincronización entre sistemas ERP y plataformas eCommerce.

Keygin permite integrar de forma automatizada **Contifico** con **WooCommerce**, asegurando consistencia en productos, inventario, precios y documentos comerciales.

---

## ¿Qué es Keygin ERP Sync Pro?

Keygin es un plugin diseñado para empresas que necesitan mantener sincronizados sus procesos administrativos y su tienda online sin intervención manual.

La sincronización incluye:

- Productos
- Stock
- Precios
- Documento eletrónico (Factura) generado a partir de una orden completada en WooCommerce

Su arquitectura está pensada para:

- Migrar/Sincronizar inventario de Contifico a WooCommerce
- Minimizar errores humanos
- Reducir tiempos operativos
- Garantizar integridad de datos
- Escalar con el crecimiento del negocio

---

## Principales características

- Sincronización bidireccional Contifico ↔ WooCommerce 
- Compatibilidad con entornos productivos  
- Registro de logs y trazabilidad  
- Configuración segura mediante credenciales API
- Preferencias de sincronización  

---

## Requisitos

Antes de comenzar, asegúrese de contar con:

- WordPress 6.9 o superior 
- WooCommerce instalado y activo 
- Cuenta activa en Contifico  
- Credenciales API válidas (Api Token, Api Key y POS)
